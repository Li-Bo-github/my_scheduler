## Overview

scheduler for preparing MS and job intern:

- col1: applying

- col3: phone
- col4: on site
- col5: offer
- col6: reject

- role
- time for (apply, phone call, on site , accept ...)
- star (optional)
- link to offical application page

## same for MS

- waitlist
- reject

with function:

- creating cluster in mongodb

- Run npm install
- Run npm run dev
